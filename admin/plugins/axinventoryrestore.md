# AxInventoryRestore

[Official AxInventoryRestore Documentation](https://docs.artillex-studios.com/axinventoryrestore.html)

## Viewing and Restoring Backups

To view a players backups, run the command `/axir view PLAYER`

From here, you can use the GUI to view and restore a player's inventory.

It is also possible to export a player's backup to a shulker box, but it is generally more favorable to restore their inventory when required.
